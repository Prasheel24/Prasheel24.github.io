Practical Implementation of an Automatic Timer on an FPGA

  • Prasheel Renkuntla
  • VHDL

This project was developed during my undergrad education in Electronics and Communication Engineering. As a part of the Verilog HDL programming course, it was designed as the final project for the semester. It uses standard vhdl libraries. It was developed and deployed with the help of Xilinx Vivado v2013.2 and was tested on a Xilinx Nexys4 FPGA board.

Dependencies: VHDL, Xilinx Vivado v2013.2

The Automatic timer uses the onboard LED to display the completion of time. The countdown is binary and the timer is set for 30 seconds. The code thus developed can be implemented on a real time system with much more complex tasks.
