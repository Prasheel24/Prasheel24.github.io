Software module for Ackermann Steering Mechanism

  • Prasheel Renkuntla, Raj Shinde
  • C++, CMake

This is a controller module for a robot that uses the Ackermann Steering Model. This controller can be implemented on any four wheeled Robot.

A controller is build for a 4 wheeled robot with an Ackermann steering to navigate through its environment. The controller consists of a PID algorithm which ensures that the velocity converges to the set point, and a Steer Algorithm that helps the robot turn.

The PID Algorithm is a control loop mechanism that calculates the error and applies correction through proportional, integral and derivative gains. The Steer Algorithm is developed to turn the robot, which is done by calculating the length of an arc inscribed between the current robot heading and target heading. The length when divided by the robot velocity gives the time for which the wheels need to be kept at angles given by Ackermann steering Model. The input to the controller will be provided by the perception model developed by the ACME Robotics. The Demonstration of the controller is given by plotting a graph that shows convergence of velocity & Heading angle to the targets with respect to time.
