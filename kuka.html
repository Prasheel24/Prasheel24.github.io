Pick, Paint and Place!

  • Prasheel Renkuntla
  • CoppeliaSim(VRep),MATLAB

This project demonstrates the use of 7 Degrees of Freedom of a KUKA LBR4 iiwa R820 arm. The aim of this project is to mark and segregate objects (pick and place) on a conveyor belt based on its size. The project is simulated in CoppeliaSim and DH table verified in MATLAB.

First, the forward and inverse kinematics were computed for the 7DoF arm. Then, using these equations the arm was simulated for different positions in its configuration space. The gripper was modelled and using its Grasp map, the force required to hold an object was calculated.

Object Sprayed

Using the FK, IK and Grasp Model of the structure, the arm is able to move throughout its configuration space covering almost 90% of it. This shows the scalability of this project to be used in any industrial environment.

Completed Task

Dependencies: CoppeliaSim(VRep),MATLAB

This project was developed for the final project in Introduction to Robot Modelling course. The arm moved easily in its configuration space to pick each of the three objects, get it sprayed and place it on the table next to it. The project can be accessed from here : GitHub Repo
